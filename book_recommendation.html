Big Data Analysis & Book Recommendation System

Using MongoDB, Hadoop, Pig, HiveQL, and HBase

Project Overview

This project involved analyzing a large dataset of book crossing information to develop a book recommendation system. The analysis was performed using big data technologies.

Key Contributions:

  • Analyzed the dataset using MapReduce filtering, organization, secondary sorting, and chaining techniques.
  • Documented and built a book recommendation system utilizing MongoDB, Hadoop, Pig, HiveQL, and HBase.

Technologies Used:

  • MongoDB
  • Hadoop
  • Pig
  • HiveQL
  • HBase
