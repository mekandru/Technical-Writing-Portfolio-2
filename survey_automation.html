Survey Automation with Salesforce & Qualtrics

Reducing Survey Time by 85%

Project Overview

This project focused on automating survey data collection and analysis by integrating Salesforce and Qualtrics. The primary goal was to reduce the time required for survey creation and feedback analysis.

Key Contributions:

  • Integrated Salesforce with Qualtrics using configurations, customizations, HTML, CSS, and JavaScript.
  • Performed response mapping and retrieved records using SOQL.
  • Automated survey data collection and analysis with a workflow rule.

Technologies Used:

  • Salesforce
  • Qualtrics
  • SOQL
  • HTML
  • CSS
  • JavaScript
